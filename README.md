# electron-vite-vue3

> 🚀 1. 基于 electron + vite + vue3 + ts + pinia的基础开发模版

> 🚀 2. 支持热更新和一键打包

> 🚀 3. 提供了一个动态修改系统主题的示例

## 构建

在构建此应用之前, 你需要安装 [Node.js](https://nodejs.org).

```bash
git clone https://github.com/UstinianMobius/electron-vite-vue3
cd electron-vite-vue3
pnpm install
pnpm electron:serve
```
